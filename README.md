# Setup

### Install

```bash
npm install
```

### Environment

```
PORT={{Port}}
MONGODB_URI={{MongoURI}}
DB_NAME={{DBName}}
```

### Import sample data

```
npm run data:import
```

### Run

```
npm start

// or

npm run dev
```

### Test users

```
// user
user1@example.com : user1_123!

// admin

```
